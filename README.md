# Angular-Portfolio
My personal portfolio webpage. Made with Angular, GSAP, and Angular Material

## Installations
### Angular
`npm install -g @angular/cli`
### NPM
`npm install`
### Angular Material
`ng add @angular/material`
### GSAP
`npm install --save gsap @types/gsap`
### Firebase
`npm install firebase @angular/fire`
## Local Deployment
`ng serve`
